# Feature: Avatar & Personalisierung

## Ready Player Me (RPM) Integration
- Nutzung des RPM Unity SDK.
- **Workflow:** 1. Nutzer öffnet Editor.
  2. Avatar wird erstellt/angepasst.
  3. RPM generiert eine `.glb` URL.
  4. Die URL wird im User-Profil in der Datenbank gespeichert.
- **Items:** Spätere Erweiterung durch eigene 3D-Assets (z.B. Finisher-Shirts), die als Layer über das RPM-Modell gelegt werden.

## Animationen
- Integration von Mixamo-Animationen (Laufen, Sprinten, Jubeln).
